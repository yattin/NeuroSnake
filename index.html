Game Instructions

Controls: Use Arrow Keys (↑ ↓ ← →) or WASD keys to move the snake.

Buttons & Settings:

  • "Start / Pause / Resume": To control game state.
  • "Toggle AI Mode": Switch between manual play and AI auto-play.
  • "Difficulty" dropdown: Select game difficulty (Easy, Medium, Hard). Affects game speed and potentially other factors. Changes apply on new game.
  • "Map Size" dropdown: Select game map size (Small, Medium, Large). Changes apply on new game or immediately if game is not running.

Objective: Eat food to grow longer. Avoid hitting walls or the snake's own body.
